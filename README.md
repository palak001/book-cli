# Book Management CLI

Command line interface for managing books. 
Uses Mongoose, Commander.js and Inquirer.js

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Create Symlink

```sh
$ npm link
```

### Commands

List Books (list or l)
```sh
$ book-cli list
```

Find Books (find or f)
```sh
$ book-cli find [NAME]
```

Add Book (add or a)
```sh
$ book-cli add
```

Update Book (update or u)
```sh
$ book-cli update [_ID]
```

Remove Book (remove or r)
```sh
$ book-cli remove [_ID]
```
